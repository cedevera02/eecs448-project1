\doxysubsection{Detailed Description}
Each ship will be placed differently and a class is needed to organize this information.

Ships will each have a size, orientation and coordinates to be stored. As the game progresses they will also take damage and be destroyed. This class helps determine that information so it can be used and altered from different classes such as game.
