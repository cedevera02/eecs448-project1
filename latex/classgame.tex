\doxysubsection{Detailed Description}
game is the driver class that handles all IO and pieces everything together.

This class is the most vital class to the project. It will take in its own parameters as well as use parts of the player, board and ship classes. Throughout the class many values for the game to progress will be created stored and used. This main logic is stored here.
